Transdirect Shipping

== Description ==

The Transdirect Shipping plugin give you access to Australia's Largest Courier and Freight Companies, and allows your clients a seamless checkout process.

The Main Features of this plugin are:

* Free to use.
* Simple Configuration.
* Great Shipping Rates from Toll Ipec, Fastway, Couriers Please, Toll Priority, Northline, Mainfreight and more.
* Seamless Checkout Process for your clients.
* Multiple Quote Options.
* Insurance Options available.

To find out more about Transdirect's service, please contact sales@transdirect.com.au.

Transdirect - Delivering Solutions.

== Installation ==

1. Make sure WooCommerce is installed and activated
2. Upload the folder `transdirect-shipping` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
